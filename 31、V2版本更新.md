## 版本二 vs 版本一 的变化

| 模块 | v1 | v2 |
|------|----|----|
| **架构** | 单线程主循环 | **三线程并发**（传感器/检测/通信） |
| **碰撞检测** | 无 | **五状态FSM**: IDLE→WEIGHTLESS→IMPACT→POST_ACC→SOS |
| **蜂鸣器** | 无 | 倒计时间歇鸣叫 |
| **按键** | 无 | GPIO中断 + 200ms消抖，取消SOS |
| **LCD显示** | 无 | ST7735 显示状态 + 倒计时 |
| **SOS上报** | 无 | 单独 SOS 主题发布含 GPS 坐标 |

### 线程分工

| 线程 | 频率 | 职责 |
|------|------|------|
| **传感器线程** | 10Hz | 读加速度→写共享区；1Hz读GNSS+温湿度+光照 |
| **碰撞检测线程** | 50ms轮询 | 读共享区加速度→驱动状态机→蜂鸣器/LCD |
| **通信线程(主)** | 100ms | MQTT上传+心跳；检测SOS标志立即发送 |

### 测试

跑一下看状态机切换和按键取消是否正常工作。测试方法：

1. 拿起板子快速晃动（模拟颠簸）— 不应触发
2. 快速下落又猛停（模拟碰撞）— 应进入 POST_ACCIDENT
3. 按键按下 — SOS取消
4. 不按键等待30秒 — 应触发 SOS
