## 4. 传感器管理模块（SensorManager）

### 4.1 模块概述

统一管理 LIS2DH12 三轴加速度计、AHT20 温湿度传感器、光敏电阻 ADC，提供线程安全的数据读取接口。

### 4.2 类结构

```
SensorManager
├── __init__()          # 初始化 I2C 总线和传感器
├── read_acceleration() # 读取三轴加速度（线程安全）
├── calc_acc_total()    # 计算合力幅值（g）
├── read_temperature_humidity() # 读取温湿度（线程安全）
├── read_light()        # 读取光敏电压
└── classify_light()    # 光照强度分类（静态方法）
```

初始化 I2C 总线和所有板载传感器。

### 4.5 calc_acc_total() 方法

#### 功能说明

计算三轴合力幅值，单位转换为 g。

#### 函数签名

```python
def calc_acc_total(self, acc):
    """
    :param acc: {'x': float, 'y': float, 'z': float} 加速度字典
    :return: 合力幅值（g）
    """
```

#### 计算公式

```
acc_total = √(x² + y² + z²) / 9.8
```

### 4.7 read_light() 方法

#### 功能说明

读取光敏电阻 ADC 值并转换为电压。

```python

    """

    """
```

#### 转换公式

```python

raw_12bit = raw_16bit >> 4

```

### 4.8 classify_light() 静态方法

#### 功能说明

根据光敏电压值分类光照强度。

#### 函数签名

```python
@staticmethod
def classify_light(voltage):
    """
    :param voltage: 电压值（V）
    :return: "亮" / "暗" / "正常"
    """
```

#### 分类规则

| 电压范围 | 返回值 |
|----------|--------|
| voltage > 2.5V | "亮" |
| voltage < 1.0V | "暗" |
| 1.0V ≤ voltage ≤ 2.5V | "正常" |

### 11.5 update() 方法

#### 功能说明

状态机主更新函数，根据当前加速度和时间戳进行状态切换。

#### 函数签名

```python
def update(self, acc_total_g, current_time_ms):
    """
    :param acc_total_g: 当前合力幅值（g）
    :param current_time_ms: 当前时间戳（毫秒）
    :return: StateResult(state, countdown)
    """
```

#### 状态流转图

```
           IDLE
             │
             │ 合力 < 0.5g 持续 100ms
             ▼
       WEIGHTLESS
             │
             │ 直接进入（不等待冲击）
             ▼
     POST_ACCIDENT ◄─────────────┐
             │                   │
             │ 30秒倒计时归零    │
             ▼                   │
       SOS_TRIGGERED ────────────┘
             │                   │
             │ 按键取消          │
             └───────────────────┘
                    ↓
                  IDLE
```

#### 状态机详细逻辑

##### 状态 0: IDLE（空闲监测）

```
输入：acc_total_g, current_time_ms

逻辑：
  if acc_total_g < FREE_FALL_THRESHOLD(0.5g):
      if freefall_start_time is None:
          freefall_start_time = current_time_ms
      elif elapsed >= FREE_FALL_HOLD_MS(100ms):
          freefall_confirmed = True
          切换到 WEIGHTLESS
  else:
      freefall_start_time = None
      freefall_confirmed = False

输出：StateResult(IDLE, 0)
```

##### 状态 1: WEIGHTLESS（失重确认）

```
输入：acc_total_g, current_time_ms

逻辑：
  countdown_start = current_time_ms
  countdown_remaining = SOS_COUNTDOWN_SEC(30)
  切换到 POST_ACCIDENT
  buzzer.beep_once(200)

输出：StateResult(POST_ACCIDENT, 30)
```

##### 状态 2: IMPACT（冲击检测，保留）

```
输入：acc_total_g, current_time_ms

逻辑：
  if 0.5 < acc_total_g < 2.0:
      countdown_start = current_time_ms
      countdown_remaining = SOS_COUNTDOWN_SEC(30)
      切换到 POST_ACCIDENT
      buzzer.beep_once(200)

输出：StateResult(POST_ACCIDENT, 30) 或 StateResult(IMPACT, 0)
```

##### 状态 3: POST_ACCIDENT（伤后静止）

```
输入：acc_total_g, current_time_ms

逻辑：
  elapsed_ms = current_time_ms - countdown_start
  remaining = SOS_COUNTDOWN_SEC - int(elapsed_ms / 1000)
  
  if remaining != countdown_remaining:
      countdown_remaining = remaining
      if remaining > 0:
          buzzer.beep_once(80)
  
  if remaining <= 0:
      切换到 SOS_TRIGGERED
      sos_sent = True
      buzzer.beep_once(500)

输出：StateResult(POST_ACCIDENT, remaining) 或 StateResult(SOS_TRIGGERED, 0)
```

##### 状态 4: SOS_TRIGGERED（SOS 已触发）

```
输入：acc_total_g, current_time_ms

逻辑：
  保持状态不变
  每 5 秒重发一次 SOS

输出：StateResult(SOS_TRIGGERED, 0)
```

### 13.4 comm_thread() 方法

#### 功能说明

单主线程分时轮询主循环，包含传感器采集、碰撞检测、MQTT 通信、LCD 显示、按键处理等所有任务。

#### 时间调度表

| 任务 | 周期 | 优先级 |
|------|------|--------|
| 加速度采集 | 100ms (10Hz) | 最高 |
| 碰撞检测 | 50ms | 高 |
| 环境数据采集 | 2s | 中 |
| GNSS 更新 | 5s | 中 |
| MQTT 数据上传 | 2s | 中 |
| LCD 显示更新 | 1s | 低 |
| MQTT 断线重连 | 30s | 低 |

#### 主循环详细逻辑

##### 阶段 1：传感器采集

```python
# 加速度（每 100ms）
if now_ms - last_acc_ms >= 100:
    acc = self.sensor.read_acceleration()
    acc_g = self.sensor.calc_acc_total(acc)
    self.shared.update_acc(acc, acc_g)
    
    # 温湿度 + 光照（每 2s）
    if now - last_env_time >= 2:
        env = self.sensor.read_temperature_humidity()
        light = self.sensor.read_light()
        self.shared.update_env(env, light)
        last_env_time = now
    
    # GNSS 更新（每 5s）
    if now - last_gps_time >= 5:
        self.gnss.update()
        last_gps_time = now
    
    last_acc_ms = now_ms
```

##### 阶段 2：碰撞检测

```python
acc_data, acc_g = self.shared.get_acc()
result = self.detector.update(acc_g, now_ms)
if result.state == CollisionState.SOS_TRIGGERED:
    self.shared.set_sos('emergency')
```

##### 阶段 3：MQTT 上传

```python
# SOS 告警发送
sos_active, _ = self.shared.has_sos()

    # 发送 MQTT SOS

# 常规数据上传（每 2s）
if now - last_upload_time >= 2:
    env, light = self.shared.get_env()
    # 构建 JSON 数据
    # 发布到 TOPIC_SENSOR
    last_upload_time = now
```

##### 阶段 4：按键处理

```python
if self.detector.sos_cancelled_flag:
    self.detector.reset_to_idle()
    self.shared.clear_sos()
    self.lcd.show_sos_cancelled()
```

## 附录：数据格式定义

### MQTT 传感器数据格式

```json
{
  "温度": "28.2℃",
  "湿度": "48.4%",
  "状态": "正常",
  "GPS定位": "北纬30°12'34.56\" 东经120°56'12.34\"",
  "速度": "6.2m/s"
}
```

### MQTT SOS 告警格式

```json
{
  "type": "sos",
  "timestamp": 1688457600,
  "latitude": 30.21,
  "longitude": 120.94,
  "speed": 0.0,
  "message": "检测到碰撞，请立即救援！"
}
```

### 状态字段取值

| 值 | 含义 |
|----|------|
| "正常" | 无异常 |
| "拿起" | 头盔被拿起 |
| "自由落体" | 检测到失重 |
| "跌倒" | 碰撞后静止 |
| "SOS" | 碰撞确认，告警已触发 |
